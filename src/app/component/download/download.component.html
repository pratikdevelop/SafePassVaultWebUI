Download for Ubuntu

Run the following command to install YourApp on Ubuntu:

  sudo add-apt-repository ppa:yourapp/ppa && sudo apt-get update && sudo apt-get install yourapp

Run with Docker

Run the following command to pull and run YourApp using Docker:

  docker pull pratikdevelop/password-app-ui:latest
  docker run -d -p 8080:8080 pratikdevelop/password-app-ui

Ensure Docker is installed on your system. Visit Docker Docs for installation instructions.
